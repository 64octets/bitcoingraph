# Blockchain parsing

*** Run bitcoin client in server mode ***

	bitcoind -daemon

*** Run server/client in RegTest mode ***

bitcoind -regtest -daemon -printtoconsole

Some test calls:

	bitcoin-cli -regtest getinfo
	bitcoin-cli -regtest getbalance
	bitcoin-cli -regtest getblockcount -> returns number of blocks
	bitcoin-cli -regtest getblockhash 0

Generate random transactions:

	https://github.com/Sigimera/bitcoin-regtest-node

Connect via cURL:

	 curl --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getinfo", "params": [] }' -H 'content-type: text/plain;' http://bitcoinrpc:pass@localhost:8332/
